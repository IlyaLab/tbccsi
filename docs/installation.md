# Installation

## Stable release

To install tbccsi, run this command in your terminal:

```sh
uv add tbccsi
```

Or if you prefer to use `pip`:

```sh
pip install tbccsi
```

## From source

The source files for tbccsi can be downloaded from the [Github repo](https://github.com/gibbsdavidl/tbccsi).

You can either clone the public repository:

```sh
git clone git://github.com/gibbsdavidl/tbccsi
```

Or download the [tarball](https://github.com/gibbsdavidl/tbccsi/tarball/master):

```sh
curl -OJL https://github.com/gibbsdavidl/tbccsi/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd tbccsi
uv pip install .
```
